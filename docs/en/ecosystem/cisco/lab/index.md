# Network Lab

Halaman ini berisi kumpulan skenario laboratorium untuk menguji teori jaringan dalam lingkungan simulasi maupun _real hardware_.

## Persiapan Environment

Sebelum memulai lab, pastikan Anda telah menyiapkan salah satu dari simulator berikut:

| Simulator               | Rekomendasi Penggunaan                         |
| :---------------------- | :--------------------------------------------- |
| **Cisco Packet Tracer** | Pemula & Persiapan CCNA dasar.                 |
| **PNETLab / EVE-NG**    | Menengah hingga Mahir (Sangat disarankan).     |
| **GNS3**                | Alternatif untuk simulasi router berbasis IOS. |

## Daftar Skenario Lab

Pilih skenario di bawah ini atau melalui menu sidebar:

### 1. Switching Dasar

- [VLAN & Trunking](/lab/vlan-trunking)
- Inter-VLAN Routing (Router-on-a-stick)

### 2. Routing Dinamis

- OSPF Single Area & Multi-area
- BGP Peering Dasar

> **Catatan Lab:** Semua topologi di sini dirancang dengan prinsip **Star Topology** atau **Mesh** untuk menjamin redundansi.
