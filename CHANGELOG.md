# Changelog

Riwayat perubahan dan pembaruan proyek Oktanetflow. Kami berkomitmen untuk terus meningkatkan platform ini dengan fitur-fitur baru dan perbaikan berkualitas.

## [v2.1.0] - 2026-01-22

### Fitur Baru

- **Ekosistem Pembelajaran Cisco**: Penambahan lab praktis untuk konfigurasi VLAN, routing, dan keamanan
- **Dukungan MikroTik**: Panduan lengkap dan lab untuk RouterOS
- **Integrasi Juniper**: Konten dasar untuk perangkat Juniper
- **Deep Dive Topics**: Bagian khusus untuk cloud networking dan network security

### Perbaikan

- Perbaikan navigasi sidebar untuk lab sections
- Optimasi performa halaman dokumentasi
- Perbaikan tautan yang rusak di beberapa halaman

### Konten

- Penambahan 15 lab praktis untuk berbagai vendor
- Pembaruan panduan konfigurasi dengan contoh terbaru
- Penambahan diagram dan ilustrasi untuk konsep kompleks

## [v2.0.0] - 2025-12-15

### Fitur Utama

- **Redesign Platform**: Migrasi ke VitePress untuk performa yang lebih baik
- **Alat Interaktif**: Penambahan kalkulator VLSM dan pembuat topologi
- **Struktur Konten**: Reorganisasi konten berdasarkan ekosistem pembelajaran
- **Bahasa Indonesia**: Transisi penuh ke bahasa Indonesia

### Perubahan Breaking

- Perubahan struktur URL untuk konsistensi
- Penghapusan konten yang tidak lagi relevan

## [v1.5.0] - 2025-10-01

### Fitur Baru

- **Panduan Dasar Jaringan**: Penambahan bagian fundamentals
- **Contoh API**: Dokumentasi untuk integrasi dengan tools eksternal
- **Responsive Design**: Optimasi untuk perangkat mobile

### Perbaikan Bug

- Perbaikan tampilan pada browser lama
- Koreksi kesalahan ketik dan informasi yang tidak akurat

## [v1.0.0] - 2025-06-01

### Rilis Awal

- **Platform Dasar**: Dokumentasi awal untuk konsep jaringan
- **Struktur Dasar**: Setup awal dengan panduan dasar
- **Komunitas Awal**: Peluncuran platform untuk feedback pengguna
