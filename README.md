# Oktanetflow

A comprehensive networking learning platform featuring tutorials, labs, and tools for Cisco, Juniper, and Mikrotik technologies.

## Features

- **Tutorials**: Step-by-step guides for routing, security, and cloud networking
- **Labs**: Hands-on exercises for VLANs, ACLs, NAT, OSPF, and more
- **Tools**: VLSM Calculator and Network Visualizer
- **Multi-Vendor Support**: Cisco, Juniper, Mikrotik ecosystems, and more
